# 逆袭算法 Comeback Algorithm

一部由 AI 创作的原创小说，讲述一个大专生用算法思维和 AI 工具逆袭人生的故事。

## 创作缘起

收到小米发的短信说 Token Plan 还有几天就要过期了，但一直没怎么用过，我写代码根本不用 mimo ，但是想到这是继小米的百万亿 tokens 扶持计划后 1 分钱续费的，又不想白白浪费了，然后我突然想起来，之前看的，大模型高考的时候，小米的 mimo-v2.5-pro 各科表现都挺好的，所以我想模型在文字上表现能力应该很强，所以我突然想到了用来写一部小说，于是就有了这部 254 章的长篇小说《逆袭算法》
