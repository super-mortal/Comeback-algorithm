# 关于本书

## 作品信息

- **书名**：Comeback Algorithm（逆袭算法）
- **类型**：现实、励志、科技
- **字数**：约60万字（254章）
- **状态**：连载中

## 故事背景

林知行，22岁，某大专计算机应用专业大三学生。成绩中下，无实习经验，无项目经验，无竞赛奖项。

在一堂普通的数据结构课上，他突然意识到：如果把人生当作一道算法题，他一直在用最低效的O(2^n)穷举法解题——穷尽所有可能性，却永远算不完。

当他遇到ChatGPT这个工具后，发现可以用O(n log n)甚至更优的算法来优化人生路径。他开始用算法思维拆解问题，用AI工具加速迭代，从课程设计到创业项目，从小城到北京，一步步找到属于自己的最优解。

## 核心主题

### 1. 算法思维与人生

将计算机科学中的核心概念应用于现实生活：
- **时间复杂度**：不同的方法，效率差异可能是数量级的
- **分治法**：把大问题拆解成小问题，各个击破
- **动态规划**：记录中间结果，避免重复计算
- **贪心算法**：在当前看来最好的选择，是否真的最优？

### 2. AI工具的正确使用

不是用AI替代思考，而是：
- 用AI辅助思考：快速生成方案、验证想法
- 用AI降低门槛：让没有资源的人也能做出成果
- 用AI加速迭代：从想法到原型，从几周缩短到几天

### 3. 普通人的逆袭

这不是天才的故事，而是：
- 如何在学历、背景、资源都不占优的情况下突破
- 如何在迷茫、焦虑、自我怀疑中找到方向
- 如何在失败、挫折、打击中继续前行

## 创作理念

这本书想告诉你的是：

**算法思维不只是写代码的工具，更是解决问题、做出选择、规划人生的方法论。**

在这个AI时代，真正重要的不是你记住了多少知识，而是你能否：
- 拆解问题
- 找到工具
- 快速迭代
- 持续优化

就像林知行说的："我们要找的不是完美的解，而是在当前约束条件下，最优的那个解。"
