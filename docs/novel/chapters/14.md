# 第十四章 场景不对

第一个找上门的客户，是个做外贸的小老板。

林知行记得很清楚，那天是周三下午，他刚写完系列帖的第二篇，正准备发出去。私信框弹出来，头像是个穿西装的中年男人，ID叫“老陈外贸”。

“林老师您好，我是做跨境的，主要卖五金配件到东南亚。最近想用AI翻译产品说明书，之前找翻译公司太贵了，一篇要五百。看到您接咨询，想问问能不能做？”

林知行点开他的主页，看了看过往动态。都是些五金配件的图片，配文是中英文混杂的广告语，语法错得离谱。

“可以，”林知行回复，“你把说明书发我，我先看看。”

老陈很快发来一份Word文档，十二页，全是螺丝、扳手、膨胀螺栓的技术参数。林知行扫了一眼，心里已经有数了。

这种翻译，ChatGPT直接就能做。甚至不需要他动手，只要把文档喂进去，调一下prompt，出来的结果比翻译公司还准。

但他还是接了。

原因很简单：他需要真实客户。

二十分钟后，林知行把翻译好的文档发回去。老陈打开一看，惊了。

“这……这也太快了吧？”

“AI辅助翻译，”林知行说，“速度快，质量也稳定。”

老陈又看了几眼，发现专业术语都对，格式也整齐，比他之前找的翻译公司强多了。

“多少钱？”

“免费。”林知行说。

“免费？”老陈愣住了，“这怎么好意思？”

“我在做咨询诊断，这次不收费。后续如果有其他需求，可以再谈。”

老陈千恩万谢地挂了电话。林知行盯着屏幕，看着那份翻译好的文档，心里没什么感觉。

太简单了。

简单到他觉得自己在浪费时间。

方小满从食堂回来，手里拎着两份炒饭。他把其中一份推到林知行面前，瞥了一眼屏幕。

“又接活了？”

“嗯。”林知行打开炒饭，扒了两口。

“多少钱？”

“免费。”

方小满的筷子停在半空：“免费？你疯了？”

“上次不是说了吗，免费诊断，先做口碑。”

“口碑也不能免费啊！”方小满急了，“你时间不要钱啊？”

林知行没接话。他继续扒饭，脑子里却在想另一件事。

这个翻译的活，他做了二十分钟。如果换成一个完全不懂AI的人，只要会用ChatGPT，也能做。可能慢一点，半小时，或者四十分钟。但结果差不多。

他的核心能力——问题拆解、架构设计、需求分析——一点都没用上。

这就像让一个外科医生去贴创可贴。

能做，但没必要。

第二个客户来得很快。

第二天上午，林知行的私信又响了。这次是个女孩，头像是奶茶店的logo，ID叫“小鹿茶饮”。

“林老师，我是开奶茶店的，最近想做一波营销活动，需要写一些朋友圈文案和小红书笔记。之前自己写太慢了，想问问AI能不能帮忙？”

林知行看了看她的主页，是个本地的奶茶品牌，三家店，粉丝不多，但内容做得挺用心。

“可以，”他说，“你把需求发我，我帮你写几条试试。”

小鹿很快发来需求：新品是草莓芝士奶盖，目标人群是18-25岁的女生，风格要可爱、俏皮、有网感。

林知行打开ChatGPT，输入prompt，三十秒后，十条文案出来了。

他挑了五条最好的，稍微改了改，发给小鹿。

小鹿看完，直接发了一串感叹号。

“天哪！这也太会写了吧！比我之前找的文案写手强多了！”

林知行看着那些感叹号，心里依然没什么感觉。

又是一个简单活。

简单到他觉得自己像个打字员。

小鹿问多少钱，林知行说免费。小鹿更激动了，连发三条语音，声音甜得发腻，说要给他介绍朋友。

林知行把手机调成静音，靠在椅背上，盯着天花板。

方小满在上铺打游戏，听到动静，探头下来：“又接活了？”

“嗯。”

“多少钱？”

“免费。”

方小满的脸垮了：“兄弟，你这样搞下去，咱俩得喝西北风。”

林知行没接话。他从抽屉里翻出一张白纸，开始写字。

方小满从床上跳下来，凑过去看。

纸上写着：

项目一：外贸翻译
耗时：20分钟
壁垒：无
替代性：高（任何人用ChatGPT都能做）
核心能力使用：0%

项目二：奶茶文案
耗时：30分钟
壁垒：无
替代性：高（任何人用ChatGPT都能做）
核心能力使用：0%

方小满看完，挠了挠头：“啥意思？”

“意思是，”林知行说，“这两个项目，谁来做都一样。”

“但客户很满意啊。”

“客户满意，是因为他们之前没见过AI。这不是我的价值，是AI的价值。”

方小满更迷糊了：“那不是一回事吗？”

林知行摇头。

他站起来，在宿舍里转了一圈。转完一圈，又转了一圈。转完三圈，他停下来看着方小满。

“我在做O(1)的工作，”他说，“但我的能力是O(n log n)的。”

方小满眨了眨眼睛：“啥意思？”

“意思是，”林知行说，“我在用牛刀杀鸡。”

方小满还是没听懂，但他看出来了——林知行不开心。

不是那种愤怒的不开心，是一种空落落的不开心。就像你费了很大劲爬上一座山，结果发现山顶是个停车场。

“那你想咋样？”方小满问。

林知行没回答。他走到窗边，看着外面灰蒙蒙的天。

十二月下旬了，天气越来越冷。远处的梧桐树叶子掉光了，只剩下光秃秃的枝丫，像伸出的手指。

他想起自己发那条动态时说的话：“我要更多真实场景。”

现在场景有了，但都是浅层场景。

翻译、写文案、做表格、生成PPT——这些活，任何一个会用ChatGPT的人都能做。他的问题拆解能力、架构设计能力、行业深度理解，全都派不上用场。

这就像一个顶级厨师，每天的工作是切菜。

能切，但不该切。

林知行转过身，走到白板前。

这块白板是他上个月买的，本来是想用来画技术架构图。现在他拿起马克笔，开始画坐标轴。

横轴：问题复杂度。
纵轴：AI能提供的价值。

方小满凑过来看。

林知行画了一条曲线。曲线从左下角开始，慢慢上升，到中间达到峰值，然后又慢慢下降。

“这是什么？”方小满问。

“价值曲线。”林知行说。

他指着左边的区域：“这一段，问题太简单。翻译、写文案、做表格——用现成的AI工具就能搞定，不需要我。”

他又指着右边的区域：“这一段，问题太复杂。比如完全自动化的无人驾驶、通用人工智能——现在的AI做不到，找我也没用。”

最后，他指着中间那段凸起的曲线：“我要的，是这一段。”

方小满皱眉：“这一段有啥特别的？”

“这一段，”林知行说，“问题刚好卡在AI能做到、但一般人做不到的区间。需要有人去拆解需求、设计方案、调优模型、处理边界case。这些，才是我的核心能力。”

方小满想了想，好像有点懂了：“所以你是说，之前的客户，都太简单了？”

“不是太简单，”林知行说，“是场景不对。”

他放下马克笔，看着白板上的曲线。

曲线的峰值在中间偏右的位置，那个位置代表的，是“有一定复杂度、但AI能解决”的问题。这些问题需要深度理解行业、拆解需求、设计方案，而不是简单地调用API。

他的三个项目——蓝天教育的智能题库、星辰教育的少儿英语系统、连锁超市的库存预测——都在这个区间。

而外贸翻译和奶茶文案，在最左边。

“那你想找什么样的客户？”方小满问。

林知行重新拿起马克笔，在白板上写了几个关键词：

数据多。
流程杂。
老板舍得花钱。
但不知道怎么花。

方小满念了一遍，还是不太明白：“啥意思？”

“意思是，”林知行说，“我需要找那些有大量数据、业务流程复杂、愿意投入资源做AI转型、但不知道从哪里下手的企业。这些企业的问题，才是真正的深度问题。”

方小满想了想：“那什么行业符合这个条件？”

林知行盯着白板，脑子里开始跑算法。

数据多——意味着要有大量的业务数据积累，不能是夫妻店那种小本生意。

流程杂——意味着业务逻辑复杂，不能是简单的买卖关系。

老板舍得花钱——意味着要有付费意愿，不能是白嫖党。

不知道怎么花——意味着需要专业指导，不能是自己就能搞定的。

这四个条件一筛选，范围就小了。

教育行业？有可能。张老板的蓝天教育，三十七家分校、上千名老师、上万名学生，数据多、流程杂，而且张老板确实舍得花钱。但他已经做过了。

医疗行业？有可能。医院的数据量大、流程复杂，而且老板（院长）确实有钱。但医院的决策链条太长，不是他一个大专生能搞定的。

金融行业？不太可能。金融的门槛太高，需要牌照、资质、合规，他连门都摸不到。

零售行业？有可能。连锁超市的数据量大、供应链复杂，而且老板确实需要AI来优化库存和选品。但他做过的那个超市项目，规模还是太小了。

林知行越想越觉得，这个筛选条件，像一个漏斗。

漏斗的入口很宽，什么行业都能往里灌。但漏斗的出口很窄，能符合条件的，没几个。

“你到底想说什么？”方小满有点不耐烦了。

林知行转过身，看着他。

“我想说，”他说，“我需要找对的客户。”

“啥叫对的客户？”

“对的客户，就是那些真正有深度问题的客户。他们的问题，不是用ChatGPT就能解决的，需要我去拆解、去设计、去优化。这些项目，才有壁垒，才有价值，才能让我积累真正的行业经验。”

方小满听完，沉默了一会儿。

然后他说：“那你咋找？”

林知行看着白板上的曲线，嘴角微微动了一下。

“我还没想好，”他说，“但我知道方向了。”

方小满点点头，虽然他还是不太懂那条曲线到底代表什么。但他看出来了，林知行的眼睛里，有光。

那种光，不是兴奋，是一种冷静的、算计的光。

像猎人在寻找猎物。

晚上，林知行躺在床上，盯着上铺的床板。

方小满已经睡着了，呼噜声很有节奏。窗外的风声呜呜地响，像是有人在哭。

他闭上眼睛，脑子里还在转。

价值曲线。

左半边的客户不需要他，右半边的客户不要他。他要找的，是中间那段。

但中间那段，到底在哪里？

他翻了个身，把被子裹紧。

十二月的夜，冷得刺骨。

他想起白天那两个客户。外贸小老板和奶茶店女孩，都很满意，都很感谢。但他知道，这不是他想要的。

他想要的，是那些真正有深度的问题。

那些让他绞尽脑汁、让他熬夜debug、让他蹲在客户现场观察一整天的问题。

那些问题，才是他的战场。

他睁开眼睛，看着黑漆漆的天花板。

明天，得想办法找找那些“对的客户”了。

怎么找？

他不知道。

但他知道，方向对了，剩下的，就是执行。

他闭上眼睛，强迫自己睡着。

梦里，他站在白板前，画着那条价值曲线。曲线的峰值在中间偏右的位置，那个位置闪着光，像一颗星星。

他伸手去抓，抓了个空。

然后他醒了。

窗外，天还没亮。

方小满的呼噜声还在继续。

林知行翻了个身，盯着墙壁。

脑子里，那条曲线还在转。

数据多、流程杂、老板舍得花钱、但不知道怎么花。

这四个条件，像四个筛选器，在他的记忆里过滤着每一个他接触过的客户。

张老板？符合，但已经做过了。
李姐？符合，但规模太小。
周主管？不符合，那是打工的，不是老板。

还有谁？

他想了很久，没想到。

但他不着急。

方向对了，剩下的，就是等。

等一个对的客户，找上门。

或者，主动去找。

他闭上眼睛，继续睡。

这一次，没有梦。
