# 第三十六章 没有退路

林知行在电脑前坐了二十分钟，才点开全国AI应用大赛的报名页面。

页面做得很花哨——深蓝色背景上飘着粒子特效，正中间一行白字："用AI改变世界，从这里开始。"下面是一个倒计时模块，红色数字跳动着：4天23小时17分。

报名截止还有五天。准确说，不到五天了。

他把页面往下拉。报名表一共四页——个人信息、参赛作品简介、团队成员、过往项目经验。前两页他填得很快，姓名、学校、联系方式、作品方向。到了学校那一栏，他顿了一下，在"湖南XX职业技术学院"几个字上停留了两秒，然后点了下一页。

团队成员。

第一个格子，他填了自己的名字。

第二个格子，方小满。

第三个格子，空着。

光标在第三个输入框里一闪一闪，像在等他。

方小满从上铺探下头来，手里攥着一袋辣条："你报了没？"

"正在填。"

"团队那栏写我了？"

"写了。"

方小满嚼了一口辣条，含糊不清地说："那第三个写谁？"

林知行没答话，盯着那个空输入框。

方小满把辣条嚼完咽下去，翻身从上铺爬下来，凑到屏幕前看了一眼。

"姜意？"他问。

"嗯。"

方小满的眉毛挑了一下："你确定？"

"她比我懂用户。"

"我不是说这个。"方小满拉过一把椅子坐下，压低了声音，"人家是互联网公司的产品总监，公众号粉丝两万多。你让她来跟咱俩组队参加比赛，她能愿意？"

林知行已经在打字了。

"不是组队，"他说，"是请她做产品顾问。"

"那你跟她说——"

消息已经发出去了。

林知行给姜意发的微信很简短："姜意，我决定参加全国AI应用大赛了。报名表上有一个团队顾问的位置，我想填你的名字。你觉得呢？"

方小满把辣条袋扔进垃圾桶，靠在椅背上等。他没说话，但手指一直在敲膝盖。

三分钟后，姜意回了消息。

"我可以给你做产品顾问，帮你把关方向和用户体验。但不挂名。"

后面跟了一段话："你想想，评委看到团队里有一个互联网公司的产品总监，第一反应是什么？会觉得你自己做不了。我不想给评委这个印象。你得让他们看到的是林知行这个人，不是林知行背后的人脉。"

林知行盯着这段话看了很久。

他回了两个字："明白。"

方小满凑过来看完聊天记录，嘴巴张了张，欲言又止，最终只是拍了拍林知行的肩膀。

"那第三个空着？"方小满问。

"空着。"

"比赛不会因为少一个人扣分吧？"

"不会。"

"那就行。"方小满站起来伸了个懒腰，"你报吧，我去买瓶水。"

他走了之后，宿舍安静下来。窗外传来篮球场上拍球的声音，一下一下，节奏均匀。

林知行把目光移回屏幕。

第三个输入框，空着。

他没有填任何人的名字。

鼠标移到页面底部的"提交"按钮上。蓝色的按钮，圆角，上面写着"确认提交报名"。

他没有立刻点下去。

不是犹豫。是一种更复杂的东西——像是站在跳板上，知道自己必须跳，但身体还需要几秒钟来做准备。

投简历的路堵死了。十七家公司，三家面试，零个offer。系统性的筛选机制，连歧视他的人都不需要出现。

创业的路天花板太低。工作室月收入两三万，受限于他一个人的产能。O(1)的瓶颈，做不出指数级增长。

专升本的路太远。两年时间，三万学费，出来之后还是鄙视链底端。

全国AI应用大赛。

不限学历。只看作品。

奖金十万。

这是最后一条路。

他想起方小满昨天说的话——"你不是为了钱。你是为了证明你行。"

方小满说得对，但也不全对。他是为了证明自己。但在证明之前，他首先得活着。

账户余额是零。母亲失业了。父亲的货车还在修车厂等着那两万块钱。

十万块。

不只是一个数字。是父亲的修车费、母亲的安心、他自己下学期的生活费，以及——一个可以被看见的机会。

林知行的手指按下鼠标。

左键，单击。

页面跳转。一行绿色的字弹出来："报名成功。您的参赛编号为AI-2024-03847。"

他的手在抖。

不是因为紧张。是因为那个编号意味着，从这一刻起，他没有退路了。

AI-2024-03847。

四万七千多人报名，最后只有前三名有奖金。概率是万分之零点六。

他想起自己曾经算过的——投简历的录用率是零。现在他换了一条路，概率从零变成了万分之零点六。

从数学角度看，这是一次巨大的提升。

从现实角度看，这依然是一个几乎不可能的数字。

他关掉报名页面，新建了一个文档。

标题：参赛方案v0.1。

文档空白，光标闪烁。

他打了一行字：

"目标：做一个让中小企业都能用得起的AI应用生成平台。"

看了十秒钟，删掉。

太泛了。姜意会说"中小企业是谁？用不起什么？平台能生成什么？"——三个追问就能把这句话拆成碎片。

重新打：

"目标：解决一个真实的、足够痛的问题。"

看了五秒钟，也删掉。

这句话是姜意说的，不是他自己的。用别人的话做自己的目标，等于没有目标。

他靠在椅背上，盯着天花板。宿舍的日光灯管嗡嗡响着，一只飞蛾绕着灯管转圈。

然后他坐直了，双手放回键盘。

这次他打得很慢，一个字一个字地敲：

"目标：证明一个大专生可以做出行业级的AI产品。"

他看着这行字。

二十三个字。没有一个字是技术术语，没有一个字是商业概念。每一个字都是他自己。

证明。大专生。行业级。AI产品。

他知道这不是一个技术目标。

技术目标应该写"开发一个基于可解释AI的应用生成引擎"或者"实现中小企业零代码AI部署方案"。那些是写给评委看的。

而这行字，是写给他自己的。

这是人生目标。

但他此刻分不清这两者的区别。

也许不需要分清。

手机震了一下。姜意又发来一条消息：

"报名了？"

林知行拍了一张报名成功的截图发过去。

姜意回："好。接下来你需要想清楚一件事——你的作品要解决谁的什么问题。不是技术问题，是人的问题。想明白了告诉我。"

林知行回："好。"

他把手机放下，重新看着屏幕上的那行字。

证明一个大专生可以做出行业级的AI产品。

他把光标移到这行字的末尾，按下回车，开始往下写。

"第一步：确定产品方向。"

"第二步：——"

光标停在那里。

他还不知道第二步是什么。但第一步已经迈出去了。

报名表提交了。编号拿到了。倒计时开始了。

四天二十三小时。

林知行把文档保存，关掉电脑。

方小满拎着两瓶矿泉水回来，递给他一瓶。

"报了？"

"报了。"

"那好。"方小满拧开瓶盖灌了一口，"饿不饿？食堂还开着。"

林知行接过水，拧开，喝了一口。矿泉水是温的，食堂的饮水机出来的，有一股塑料味。

"走吧。"他说。

两人出了宿舍楼。天快黑了，路灯还没亮，校园里的广播在放一首听不清歌词的歌。几个学生骑着自行车从他们身边掠过，笑声在空气里散开。

林知行走在方小满旁边，手里攥着那瓶水。

他想起父亲跑完长途回来的样子——满身灰尘，眼圈发黑，坐在门槛上抽一根烟。父亲从来不说累。但他的背会弯下去，比平时弯一些。

现在轮到他了。

他也不会说累。但他知道自己的背也在弯。

不过没关系。

背弯着也能走。

走到食堂门口，方小满忽然停住脚步，回头看他。

"知行。"

"嗯？"

"你刚才提交报名表的时候，我在门口看到了。"

林知行愣了一下："你不是去买水了？"

"买水要下楼，下楼要经过你背后。"方小满的语气很随意，"我看到你的手在抖。"

林知行没接话。

方小满转过身，继续往食堂走。

"没事，"他说，"手抖就抖吧。手抖说明你知道自己在干什么。不怕的人才是真的蠢。"

林知行跟在他后面，走进食堂。

食堂快关门了，灯关了一半，只剩下打饭窗口附近还亮着。阿姨在擦桌子，看到他们进来，喊了一声："只剩番茄炒蛋和土豆丝了！"

"行。"方小满端了两个餐盘过来。

两人坐在角落的位置，吃着番茄炒蛋和土豆丝。食堂的米饭已经有点凉了，番茄炒蛋偏甜，土豆丝切得粗细不一。

但林知行吃得很认真。

因为他不知道下一顿还能不能吃上食堂。

方小满吃着饭，忽然冒出一句："你说的那个比赛，最后几天报名来着？"

"还有四天多。不是报名截止，是报名截止了要交作品简介。"

"那咱们得抓紧。"

"嗯。"

"你有想法了吗？"

林知行嚼了一口土豆丝，咽下去。

"有个方向，"他说，"但还没想透。"

"什么方向？"

"姜意说的那句话——解决一个具体的、真实的、足够痛的问题。"

方小满想了想："那咱们最熟的问题是什么？"

"排课。"

"那就做排课。"

"太窄了。比赛要的是通用性。"

方小满夹了一筷子番茄炒蛋，边嚼边想。

"那什么问题既够痛又够通用？"

林知行放下筷子。

这个问题他想了两天了，但此刻方小满一问，他脑子里忽然浮现出一个画面——

母亲在电话里说"超市不让我干了"时的表情。他没看到，但他能想象。

智能收银系统取代了十二个收银员中的八个。那些被取代的人，像他母亲一样，四十六岁，没有学历，没有技能，被一台机器和一套算法推到了生活的边缘。

他们不懂AI。他们被AI淘汰了。

但如果反过来呢？

如果他们能用AI呢？

如果有一个工具，让一个不懂技术的超市收银员也能用AI来分析自己的处境、找到新的出路？

他摇了摇头。太理想化了。

"我还没想好。"他说，"但我知道一件事。"

"什么？"

"这个作品不能只证明技术厉害。它得解决一个真实的问题。如果解决不了，技术再厉害也没用。"

方小满点头："你说得对。那你慢慢想，我负责——你让我干啥我干啥。"

林知行看着他，嘴角动了一下。

这个动作大概算是笑了。

吃完饭，两人走出食堂。天彻底黑了，路灯亮了，橙黄色的光洒在水泥路面上。远处的教学楼只剩几个窗口还亮着灯。

回到宿舍，林知行打开电脑，重新看那个参赛方案文档。

v0.1。

"目标：证明一个大专生可以做出行业级的AI产品。"

"第一步：确定产品方向。"

"第二步：——"

他在"第二步"后面打了几个字，又删掉了。

然后他新建了一个文档，标题是"参赛方案v0.2"。

这次他没有写目标。他在文档里列了一张清单——

"我能做什么？"
- 排课系统（有完整案例）
- AI解释层（有数据验证）
- 用户调研方法论（有姜意的报告）

"我缺什么？"
- 一个足够痛的通用问题
- 一个能让评委眼前一亮的产品形态
- 一个能证明技术深度的核心功能

"我有什么？"
- 沈渡可以递作品简介给评委
- 张老板的行业案例
- 姜意的产品顾问

他看着这张清单，意识到一个事实：他有的东西很多，但缺的那个"通用问题"才是关键。

就像写代码——函数库再全，如果需求没搞清楚，写出来的东西也是废的。

他关掉v0.2，回到v0.1。

目标那一行还在。

"证明一个大专生可以做出行业级的AI产品。"

他盯着看了很久，然后在后面加了一行字：

"方法：找到一个真实的、足够痛的问题，用技术解决它。"

两行字并排放在文档开头。

第一行是目的，第二行是路径。

目的和路径之间，是四天二十三小时的倒计时。

林知行关掉文档，关掉电脑，上床躺着。

方小满在上铺刷手机，屏幕的光从床板缝隙里漏下来。

"知行。"

"嗯？"

"你刚才在食堂说的那个想法——让不懂技术的人也能用AI——我觉得挺好的。"

"太理想化了。"

"理想怎么了？"方小满翻了个身，"理想又不要钱。"

林知行没接话。

方小满继续说："再说了，你做的事哪一件不理想化？大专生做AI创业、给教育机构写排课系统、参加全国比赛——哪一件听起来不像是吹牛？但你都做成了。"

林知行盯着天花板，听着这些话。

方小满最后说了一句："你就是那种把吹的牛变成现实的人。"

然后他关了手机，上铺的光消失了。

宿舍暗下来。只有窗外路灯的光透进来，在地面上拉出一个长长的光斑。

林知行闭上眼睛。

脑子里还在转。

v0.1的目标。四天的倒计时。账户里的零。母亲的声音。父亲的货车。方小满的话。

这些信息像一个待处理的任务队列，排着队等他处理。

但他现在处理不了。

他需要先把最关键的那件事想清楚——作品做什么。

这个问题的答案不在代码里，不在数据里，不在任何算法能推导的地方。

它在一个他还没找到的地方。

但他知道，他必须找到。

因为AI-2024-03847这个编号已经生成了。报名费零元。代价是——他把最后一条退路堵死了。

从明天开始，他要做的事情只有一件。

往前走。

（本章完）
