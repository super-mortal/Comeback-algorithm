# 第三十五章 出账

手机响的时候，林知行正盯着屏幕上那三百行代码。

是母亲的电话。

林知行看了一眼时间——下午四点十七分。母亲平时很少这个点打电话，她应该在超市上班。他心里闪过一丝不安，接起电话。

“妈？”

“知行啊……”母亲的声音有些犹豫，不像平时那样干脆。

“怎么了？”

电话那头沉默了几秒。林知行听到母亲深吸了一口气。

“妈跟你说个事，你别着急。”

“你说。”

“超市……超市不让我干了。”

林知行的手指停在键盘上。

“什么意思？”

“就是……”母亲的声音低下去，“超市装了那个自助结账机，收银员从十二个减到四个。我是被减掉的那八个里的一个。”

林知行没有说话。

母亲继续说：“公司给了一个月的遣散费，说让我们自己另找工作。但是知行，妈今年四十六了，别的超市也不要我这么大的……”

她的声音越来越小，最后变成了一声叹息。

林知行盯着屏幕上的代码，那些字符突然变得陌生。

智能收银系统。

自助结账机。

他靠AI起家，靠AI赚钱，靠AI在技术社区建立了口碑。现在，AI夺走了他母亲的工作。

“妈，你别急。”林知行说，声音比自己预想的要平静，“我这边有办法。”

“你能有什么办法？你还在上学……”

“我有项目收入，”林知行说，“张老板那边的尾款还没结，等结了我转给你。”

母亲没有接话。

林知行知道她在想什么——她不想成为儿子的负担。她是母亲，她应该供养儿子，而不是反过来。

“还有个事，”母亲的声音更低了，“你爸的货车……修车厂说要大修，不然没法跑了。要两万块。”

两万块。

林知行的心沉了一下。

“家里还有多少？”他问。

“你爸跑了两趟长途，攒了一万二。加上我的遣散费，八千。总共两万。”

“那不是刚好吗？”

“可是……”母亲顿了一下，“你爸说，修车要两万，但家里总得留点钱应急。他想跟你商量，看能不能先借点。”

林知行闭上眼睛。

他打开手机银行，看了一眼余额。

六千四百块。

这是他之前攒的，加上最近几个小项目的收入。张老板的尾款两万一千块还没到账，合同写的是项目验收后三十个工作日内支付。现在才过了不到两周。

“妈，我这里有六千四。”林知行说，“我先转给你们，修车要紧。”

“那你自己怎么办？你还要吃饭、交资料费……”

“我有办法。”林知行说，“方小满那边能借我点，你别担心。”

这是他今天第二次撒谎。

方小满的钱是专升本的学费，三千块。他不能动。

母亲在电话那头沉默了很久，然后说：“知行，妈妈没用。”

“妈你别说这个。”林知行的声音有点紧，“是我没用。我应该早点赚到钱的。”

“你已经很厉害了，”母亲说，“张老板那个项目，你做得很好。你爸跟邻居说的时候，眼睛都亮了。”

林知行没有说话。

“但是知行，”母亲的声音突然认真起来，“你那个AI，能不能帮你爸想想办法？他跑了二十年货车，除了开车什么都不会。现在货运行情不好，要是车修好了还是拉不到活，怎么办？”

林知行张了张嘴，没说出话。

AI能帮张老板排课，能帮淘宝卖家写文案，能帮教育机构做智能题库。但它帮不了一个五十二岁的货车司机。

“妈，这个我再想想。”他说，“你先别急，我马上转钱给你。”

挂了电话，林知行坐在椅子上，盯着天花板。

他用手机银行操作转账。六千四百块，全部转出。

转账成功的提示跳出来，余额变成零。

零。

他想起自己刚用AI赚到第一笔钱的时候，给父亲转了五十块。父亲过了三个小时才回了一个字：嗯。

现在他转了六千四百块，父亲不会知道，因为这是转给母亲的。母亲会告诉父亲，但父亲只会说一句：别乱花钱。

林知行关掉手机银行，打开一个Excel表格。

他开始算账。

母亲的遣散费：八千块。

父亲的积蓄：一万二。

修车费：两万块。

缺口：零。

但是，修完车之后呢？

父亲跑一趟长途，油费、过路费、食宿，成本大概一千五。收入三千到四千。净利润一千五到两千五。

一个月跑四趟，收入六千到一万。

但是，这四趟不是每个月都能跑满的。淡季的时候，可能一个月只有一两趟。

母亲失业了，没有收入。

他自己，账户余额为零。张老板的尾款两万一千块，不知道什么时候到账。下一个小项目还不知道在哪里。

林知行在Excel表格里写下几行数字：

家庭月收入（乐观估计）：父亲8000 + 我4000 = 12000

家庭月收入（悲观估计）：父亲3000 + 我0 = 3000

家庭月支出：房租（老家）800 + 生活费3000 + 我的生活费1500 + 专升本资料费500 = 5800

他盯着这些数字，第一次感到算法思维的无力。

这些数字能算出来，但解法呢？

父亲不能不跑车，那是他唯一的技能。母亲四十六岁，没有学历，超市的工作已经是最适合她的了。他自己，还在上学，能做的只有接AI项目。

但是AI项目越来越难接了。

三个月前，兼职市场的要求是“会用ChatGPT”。现在变成了“有AI项目经验者优先，本科以上”。连兼职市场都在学历筛选。

林知行关掉Excel，靠在椅背上。

窗外的天色暗下来了。宿舍楼下的路灯亮了，橙黄色的光透过窗帘的缝隙照进来。

他想起自己算过的那笔账——父亲跑一趟长途挣两千八，自己做一个项目挣五千。看起来他赚得更多。但父亲的两千八是确定的，自己的五千不知道下一个在哪里。

确定性。

这是父亲那代人最看重的东西，也是他最缺的东西。

手机震了一下。微信消息。

方小满：“考完了！！！”

林知行回了一个字：“好。”

方小满：“你咋了？”

林知行：“没事。”

方小满：“你肯定有事。你在宿舍吗？我回来了。”

林知行没有回复。

他坐在椅子上，盯着电脑屏幕。屏幕已经黑了，映出他自己的脸——瘦，戴眼镜，下巴上有一圈淡淡的胡茬。

他想起姜意昨天的话：“你要解决一个具体的、真实的、足够痛的问题。”

他现在面对的，就是一个具体的、真实的、足够痛的问题。

但他解决不了。

AI解决不了。

他的算法思维解决不了。

门被推开了，方小满走进来，背着书包，脸上带着考完试的轻松。

“知行，我跟你说——”他看到林知行的脸色，停住了。

林知行转过头，看着他。

方小满放下书包，走过来，在他对面坐下。

“怎么了？”

林知行张了张嘴，想说没事，但没说出口。

方小满没有追问。他站起来，拉了把椅子，坐在林知行旁边。

两人沉默了很久。

宿舍楼下的车流声隐约传来，偶尔有学生的笑声飘过。

“你是不是缺钱？”方小满问。

林知行没有说话。

方小满从口袋里掏出手机，打开支付宝：“我这里还有三千，你先拿去。”

林知行摇头：“你那三千是专升本的学费。”

“学费可以再挣，”方小满说，“兄弟只有一个。”

林知行想哭，但忍住了。

他看着方小满的脸——圆脸，眼睛不大，但很亮。这张脸他看了三年，从大一到现在。方小满是唯一一个从不因为学历或成绩看不起他的人。

“小满，”林知行说，“我决定参加那个比赛。”

方小满愣了一下：“哪个比赛？”

“全国AI应用大赛。奖金十万。”

方小满的眼睛亮了：“真的？那好啊！你肯定能拿奖！”

林知行摇头：“我不是为了拿奖。”

“那你为什么？”

“为了钱。”

方小满看着他，沉默了几秒。

然后他说：“你骗人。”

林知行没有说话。

方小满继续说：“你不是为了钱。你是为了证明你行。”

林知行想反驳，但发现自己说不出口。

因为方小满说对了。

他参加比赛，确实是为了十万奖金。但更深处，他是为了证明自己。证明一个大专生可以做出行业级的AI产品。证明学历不是天花板。证明他走的路是对的。

这些证明，比十万块更重要。

但他现在不能承认。

因为承认了，就意味着他在用家里的经济危机给自己找借口。

“我真的是为了钱。”林知行说。

方小满摇头：“你要是真为了钱，你会去接更多的小项目，而不是去赌一个比赛。比赛只有前三名有奖金，万一拿不到呢？”

林知行没有回答。

方小满站起来，拍了拍他的肩膀：“不管为了什么，你决定了就好。需要我做什么，你说。”

林知行看着他，点了点头。

方小满回到自己的位置，放下书包，开始收拾东西。

林知行坐在椅子上，盯着黑掉的电脑屏幕。

十万块。

如果能拿到一等奖，家里所有的问题都能解决。父亲的修车费，母亲的遣散费，他自己下学期的生活费。

但是，他连参赛作品做什么都不知道。

报名截止还有五天。

他想起姜意昨天的话：“标准化产品和定制项目是两回事。”

他想起沈渡的话：“比赛还是要看作品本身。”

他想起父亲的话：“别乱花钱。”

他想起母亲的话：“妈妈没用。”

这些话在他脑子里转，像一个循环的死结。

然后，他想起了一句话。

是方小满刚才说的：“你不是为了钱。你是为了证明你行。”

证明你行。

林知行打开电脑，新建了一个文档。

标题是：参赛方案v0.1。

他在文档里写了一行字：

目标：证明一个大专生可以做出行业级的AI产品。

然后他删掉了。

重新写：

目标：做一个让中小企业都能用得起的AI应用生成平台。

又删掉了。

最后，他写了一行字：

目标：解决一个真实的问题。

他看着这行字，想起这是他第四次写这行字。前三次他都删掉了。

这次，他没有删。

因为他突然明白了一件事——

他要解决的真实问题，不是技术问题，不是商业问题，而是他自己的问题。

他要证明的，不是大专生能做出什么，而是他林知行能做出什么。

这两者，有本质的区别。

前者是标签，后者是人。

他以前一直在和标签战斗，现在他要开始和自己战斗。

林知行关掉文档，关掉电脑，上床睡觉。

窗外的天已经全黑了。路灯的光透过窗帘，在天花板上投下一片昏黄。

他闭上眼睛，脑子里还在转。

父亲的货车，母亲的工作，自己的比赛。

三个变量，一个方程组。

解法未知。

但他知道一件事——他必须解出来。

不管用什么算法。

（本章完）
