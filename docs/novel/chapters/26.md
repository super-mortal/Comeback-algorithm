# 第二十六章 变量与常量

宿舍里只有键盘敲击声和游戏音效。

方小满在上铺打排位，嘴里偶尔蹦出一句"nice"或者"我靠"。林知行坐在下面，笔记本电脑开着，屏幕上不是代码，而是一个新建的文本文档。

光标在第一行闪烁。

他打了几个字，又删了。再打，再删。

最后他放弃了，把文档最小化，目光落在桌面上的三样东西上。

左边是一张收据——张老板的项目尾款，两万一千块，昨天下午到账的。打印出来的银行回单，纸有点皱，被他从口袋里掏出来时折过。这个数字在他二十二年的人生里从未出现过。他父亲跑一趟长途，四天，两千八百块。这两万一千块，是他父亲不吃不喝跑七个多趟才能赚到的钱。

中间是一篇报道——《让每一个学生都能被看见：一个大专生的AI教育实验》。手机屏幕上还开着那个页面，标题用的是他说的那句话。他盯着"大专生"三个字看了几秒，又看了看下面的评论区，有人说"这才是AI落地的正确姿势"，有人说"92%的数据有没有第三方验证"。他把评论区关了。

右边是那封邮件。

沈渡。灵犀科技。教育AI项目。

"可以见面吗？"

他把邮件又点开看了一遍。正文只有一句话，没有寒暄，没有吹捧，没有"久仰大名"之类的客套。这种风格反而让他更拿不准——如果对方写一堆恭维话，他可以判断为套路；但这么简洁，要么是真的很忙，要么是真的不在意，要么是吃准了这种简洁本身就会让人多想。

"你又盯着那封邮件看？"方小满从上铺探出头。

林知行把邮件窗口关了。"没。"

"你看了至少五遍了。"方小满翻身坐起来，"从昨天晚上就开始看。你要不要直接回他？"

"我在想。"

"想什么？"

"想他到底想要什么。"

方小满从梯子上爬下来，光着脚站在地上，看了一眼林知行的屏幕，又看了一眼桌上的收据和手机上的报道。

"你现在有项目有钱有口碑还有美女搭档，"他说，"大专生混成你这样的不多了吧。"

林知行没接话。

"我说真的，"方小满在他旁边坐下，"你看看你这半年——第一笔钱是四百五，现在两万一千块。张老板逢人就推荐你，那个记者写了那么长一篇报道，还有姜意……"

"跟姜意有什么关系？"

"人家推荐你接受采访，你还欠人家一顿饭呢。"方小满说，"你现在是'小林AI工作室'的技术核心，有案例库，有行业人脉，还有媒体曝光。你算算，大专生里有几个能做到这一步？"

林知行还是没说话。

方小满看了他一眼："你该不会在想沈渡那个事吧？"

"嗯。"

"你想去见他？"

"我在算。"

"算什么？"

"算这个决定的复杂度。"

方小满翻了个白眼："你又来了。"

林知行打开那个文本文档，光标还在第一行。他这次没有犹豫，开始打字：

```
def life_algorithm():
    # 当前状态
    projects = ["张老板排课系统"]  # 已完成项目
    income = 21000 + 414  # 当前余额
    reputation = ["技术社区发帖", "行业媒体报道", "张老板推荐"]  # 行业影响力
    network = ["方小满", "张老板", "刘主管", "姜意"]  # 人脉
    invitation = {
        "from": "沈渡",
        "company": "灵犀科技",
        "role": "未明确",
        "background": "15年AI经验，前大厂VP"
    }
    
    # 决策变量
    known_factors = {
        "项目经验": True,
        "行业口碑": True,
        "媒体曝光": True,
        "经济基础": "初步积累"
    }
    
    unknown_factors = {
        "沈渡的真实意图": None,
        "灵犀科技的实际状况": None,
        "自己能提供什么价值": None,
        "这次机会的真实门槛": None
    }
    
    # 决策逻辑
    if unknown_factors["沈渡的真实意图"] == None:
        # 无法计算
        return "需要更多信息"
    elif invitation["background"].includes("大厂VP"):
        # 高价值机会
        return "accept"
    else:
        # 低优先级
        return "hold"
```

他写完了，盯着屏幕。

方小满凑过来看了一眼："这什么？"

"人生算法。"

"我看不懂。"方小满说，"但你最后那个'return'是什么意思？"

"需要更多信息。"林知行说，"因为我算不清那个变量。"

"什么变量？"

"沈渡到底想要什么。"

方小满想了想："你直接问他不就行了？"

"问什么？"

"问他为什么找你啊。"

林知行摇头："他不会直接说的。如果他的意图是好的，他会用行动证明；如果意图不好，他更不会直接说。问了也没用。"

"那你就一直在这算？"

"我在找他的模式。"林知行说，"他做了十五年AI，从大厂出来创业，专注教育赛道。他看了我的报道，主动发邮件。这些是已知的。"

"然后呢？"

"然后我要算他缺什么。"林知行说，"一个做了十五年的技术VP，为什么要在创业两年后主动联系一个大专生？要么他缺人，要么他缺某种东西——某种他在我身上看到的东西。"

方小满听了一会儿，忽然笑了："你有没有想过，也许他就是觉得你牛逼？"

"不可能。"林知行说，"他比我牛逼一百倍。十五年经验，大厂VP，创业公司创始人。我算什么？一个大专生，做了几个小项目，上了一次新闻。他凭什么觉得我牛逼？"

"那你说他图什么？"

"我不知道。"林知行说，"所以我算不出来。"

他把文本文档关了，靠在椅背上。

窗外是傍晚，五月的空气里有栀子花的味道。宿舍楼下有人在打球，喊叫声传上来，断断续续的。林知行的手机亮了一下，是银行APP的推送——账户余额21,414.00元。

他看了一眼，把手机翻过来扣在桌上。

"方小满。"他说。

"嗯？"

"你说，如果这个机会是真的——我是说，如果沈渡真的想跟我合作——我应该去吗？"

方小满认真想了一下："你为什么不去？"

"我怕。"

"怕什么？"

"怕我配不上。"林知行说，"怕我到了那个层面，发现自己什么都不是。怕我见了沈渡，被他几句话就看穿了。怕我发现自己的那些东西——案例库、方法论、排课系统——在真正的专业人士眼里，根本不算什么。"

方小满没说话。

"我之前也怕过。"林知行说，"第一次见张老板的时候怕，第一次接受采访的时候怕。但那些怕，我能算出来——我知道问题是什么，方案是什么，最坏结果是什么。但这次不一样。这次我面对的不是一个具体的问题，而是一个人。一个比我厉害一百倍的人。"

"所以你就不去了？"

"我没说不去。"林知行说，"我在算。"

"你算了多久了？"

"从昨天晚上到现在。"

"算出来了吗？"

"没有。"

方小满站起来，走到窗边，往外看了一眼。

"你知道吗，"他说，"我觉得你最大的问题就是太能算了。"

"什么意思？"

"你什么都想算清楚，但有些东西算不清楚的。"方小满说，"就像你追姜意……"

"我没追姜意。"

"行，就像你和姜意合作。"方小满说，"你算过吗？你算过你跟她合作能带来什么价值吗？你算过她推荐你接受采访对你有多少好处吗？"

林知行摇头。

"那不就对了。"方小满说，"有些事情，你算不清楚，但你还是做了。因为你觉得应该做。"

"那不一样。"

"有什么不一样？"

"和姜意合作，我能提供技术，她能提供产品视角。这是互补的。但和沈渡……我不知道我能提供什么。"

方小满转过身，看着他："你有没有想过，也许你不需要提供什么？也许他就是想见见你？"

"见我干什么？"

"我怎么知道。"方小满说，"但你不去，就永远不知道。"

林知行没接话。

他重新打开那个文本文档，把光标拉到最后一行。在`return "需要更多信息"`下面，他又加了一段：

```
# 但是
# 有些变量的值，只有走进去才能知道
# 有些函数的返回值，只有调用了才能得到
# 有些路，只有走了才知道通不通
```

他写完了，盯着屏幕看了很久。

然后他把文档关了，打开邮箱，找到沈渡那封邮件。

光标在回复框里闪烁。

他打了几个字：

"沈总您好，感谢您的邮件。"

他看了看，删了。太客气了，像求职邮件。

又打了几个字：

"可以，您定时间。"

他看了看，又删了。太随意了，不尊重对方的身份。

第三次，他打了这段话：

"沈总您好，感谢您的邮件。我最近确实在思考教育AI方向的一些问题，很期待能和您交流。请问您什么时间方便？"

他盯着这段话，手指悬在发送键上。

"方小满。"他说。

"干嘛？"

"你说我该不该加'期待'这个词？会不会显得太主动了？"

方小满从上铺探出头，看了他一眼："你加不加都行。别磨叽了。"

林知行深吸一口气，把"很期待"改成了"希望"。

然后他点了发送。

邮件发出去的一瞬间，他的心跳快了一拍。

他不知道自己是在走向一个更大的舞台，还是走进一个更大的笼子。但他知道，他不能一直站在门口算复杂度。

有些变量，只有走过去才能知道它的值。

他靠在椅背上，盯着天花板。方小满在上铺继续打游戏，偶尔发出一声欢呼。宿舍里恢复了之前的节奏——键盘声、游戏音效、风扇转动的声音。

林知行的手机亮了一下，是沈渡的回复。

只有一行字：

"下周三下午三点，国贸三期大堂咖啡厅。"

林知行盯着那行字，手指有点发麻。

国贸三期。北京。

他从没去过北京。

他打开手机银行，看了一眼余额——21,414.00元。如果去北京，高铁票大概五百块，住宿一晚大概三四百，加上吃饭和交通，至少要一千块。

他想了想，又打开了备忘录。那行字还在：

"可解释AI——排课系统只是起点。"

他在后面加了一行：

"沈渡——灵犀科技——教育AI项目——下周三。"

然后他关了手机，躺在床上。

窗外已经暗下来了，宿舍的灯没开，只有电脑屏幕的光映在墙上。方小满在上铺翻了个身，忽然说：

"你要去北京？"

"嗯。"

"什么时候？"

"下周三。"

"要我陪你吗？"

林知行想了想："不用。你忙你的。"

"行。"方小满说，"那你早点买票，别到时候买不到。"

"嗯。"

方小满又翻了个身，安静了一会儿。

"林知行。"他说。

"嗯？"

"你记不记得大一的时候，我在床板上刻了一行字？"

"记得。"

"你还记得是什么字吗？"

林知行没说话。他不需要记，那行字就在他头顶的床板上，他每天躺下都能看到。

"干就完了。"方小满说。

林知行盯着上铺的床板，那行字在黑暗里看不太清，但他知道它在那里。

"嗯。"他说。

"有些事情，算不清楚的。"方小满说，"你算来算去，最后还是要干。"

林知行没接话。

他闭上眼睛，脑子里还在转。沈渡、灵犀科技、国贸三期、下周三。这些词在他脑子里排列组合，像一段代码在运行。但他知道，这段代码的输出，不是靠算出来的。

是靠干出来的。

他想起父亲上次打电话说的那句话——"别乱花钱"。

他看了看手机——余额21,414元。比开学时多了五十倍，但离改变命运还差得远。

他关了手机，躺在床上，盯着上铺的床板。

那上面有一行字，是方小满大一时候刻的：

"干就完了。"

林知行想，也许人生算法的最后一步，不是算出来，是干出来。

（本章完）
