# 学习路线

根据你的基础和目标，我们提供了不同的学习路线建议。

## 初学者路线（0-3个月）

### 阶段一：数据结构基础
1. 数组和字符串
2. 链表
3. 栈和队列
4. 哈希表

### 阶段二：基础算法
1. 排序算法
2. 二分查找
3. 双指针技巧
4. 滑动窗口

### 阶段三：树的基础
1. 二叉树遍历
2. 二叉搜索树
3. 树的基本操作

## 进阶路线（3-6个月）

### 阶段一：高级数据结构
1. 堆
2. 图的表示和遍历
3. 并查集
4. 前缀树（Trie）

### 阶段二：高级算法
1. 动态规划入门
2. 贪心算法
3. 回溯算法
4. 深度优先搜索（DFS）
5. 广度优先搜索（BFS）

### 阶段三：专题突破
1. 字符串算法
2. 位运算
3. 数学问题

## 面试冲刺路线（1-2个月）

### 重点专题
1. 高频算法题
2. 经典动态规划
3. 图算法进阶
4. 系统设计基础

### 刷题策略
- 每天 2-3 道题
- 按照类型集中练习
- 定期复习已做题目
- 总结解题模板

## 学习建议

1. **不要贪多**：每个阶段扎实学习，不要急于求成
2. **动手实践**：每个算法都要自己实现一遍
3. **总结归纳**：整理自己的解题模板和思路
4. **定期复习**：算法需要反复练习才能熟练

选择适合自己的路线，开始学习吧！
