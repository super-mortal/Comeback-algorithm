# 如何使用

本文档将帮助你快速上手使用本站点。

## 导航结构

### 学习指南

包含算法学习的基础知识和学习方法：
- 时间复杂度和空间复杂度
- 常用算法技巧
- 学习路线建议

### 算法分类

按照数据结构和算法类型分类：
- **数组**：双指针、滑动窗口等
- **链表**：反转、快慢指针等
- **树**：遍历、二叉搜索树等
- **动态规划**：经典 DP 问题
- **回溯算法**：排列组合、DFS 等

### 题解

提供详细的算法题目解答，包括：
- 题目描述
- 解题思路
- 代码实现
- 复杂度分析
- 相关题目

## 使用技巧

### 搜索功能

使用页面右上角的搜索框快速查找内容。

### 代码复制

所有代码块都支持一键复制，点击代码块右上角的复制按钮即可。

### 目录导航

每个页面右侧都有目录导航，方便快速跳转到感兴趣的章节。
