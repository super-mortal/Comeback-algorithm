# 树

树是一种非线性数据结构，在算法中有广泛应用。

## 什么是树

树是由节点和边组成的层次结构，有一个根节点，每个节点可以有多个子节点。

## 基本概念

- **根节点**：树的顶端节点
- **叶子节点**：没有子节点的节点
- **深度**：从根到该节点的路径长度
- **高度**：从该节点到叶子节点的最长路径

## 二叉树

每个节点最多有两个子节点的树。

```javascript
class TreeNode {
  constructor(val, left = null, right = null) {
    this.val = val;
    this.left = left;
    this.right = right;
  }
}
```

## 常见类型

1. **完全二叉树**：除最后一层外，其他层都满，最后一层从左到右连续
2. **满二叉树**：每个节点要么有两个子节点，要么没有子节点
3. **二叉搜索树**：左子树 < 根 < 右子树
4. **平衡二叉树**：左右子树高度差不超过 1
