# 数组

数组是最基础、最常用的数据结构之一。

## 什么是数组

数组是一种线性数据结构，它用一组连续的内存空间来存储一组具有相同类型的数据。

## 数组的特点

- **随机访问**：可以通过索引在 O(1) 时间内访问任意元素
- **连续存储**：元素在内存中连续存储
- **固定大小**：大多数语言中，数组创建后大小固定

## 常见操作

| 操作 | 时间复杂度 |
|------|-----------|
| 访问元素 | O(1) |
| 查找元素 | O(n) |
| 插入元素 | O(n) |
| 删除元素 | O(n) |
