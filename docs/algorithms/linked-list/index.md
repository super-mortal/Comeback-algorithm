# 链表

链表是一种重要的线性数据结构。

## 什么是链表

链表是通过指针将一组零散的内存块串联起来使用的数据结构。

## 链表的特点

- **动态大小**：可以动态扩展和收缩
- **非连续存储**：节点在内存中不连续
- **插入删除高效**：O(1) 时间复杂度（已知位置）
- **随机访问慢**：需要 O(n) 时间

## 常见类型

1. **单链表**：每个节点只有一个指向下一个节点的指针
2. **双向链表**：每个节点有两个指针，分别指向前后节点
3. **循环链表**：尾节点指向头节点

## 基本节点定义

```javascript
class ListNode {
  constructor(val, next = null) {
    this.val = val;
    this.next = next;
  }
}
```
